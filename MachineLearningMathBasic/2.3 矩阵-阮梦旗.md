# 2-3 矩阵
### 矩阵定义
矩阵就是一个矩形阵列，阵列尺寸为m * n。
矩阵中有m * n个元素，元素可以是数字、符号或者数学式。

### 矩阵的运算
加减运算
>当A、B两个矩阵行数和列数相同时，这两个矩阵可直接进行加减运算，即矩阵中相同地址的元素直接进行加减运算。

乘法运算
> 当A的列数等于B行数时，A（i * j）可乘以B(j * k)，记为A * B(不可更换位置)，得到一个新的矩阵C（i * k）。
>
> >在进行乘法运算时，矩阵的运算满足**结合律**和**分配率**，但是**不满足**交换律。
> >结合律  : (A+B)+C = A+(B+C)
> >分配律 : A * (B + C) = (A * B) + (A * C)
> >_不满足交换律 A * B != B * A_
